1802. 有界数组中指定下标处的最大值

难度中等42

给你三个正整数 n、index 和 maxSum 。你需要构造一个同时满足下述所有条件的数组 nums（下标 从 0 开始 计数）：

  • nums.length == n
  • nums[i] 是 正整数 ，其中 0 <= i < n
  • abs(nums[i] - nums[i+1]) <= 1 ，其中 0 <= i < n-1
  • nums 中所有元素之和不超过 maxSum
  • nums[index] 的值被 最大化

返回你所构造的数组中的 nums[index] 。

注意：abs(x) 等于 x 的前提是 x >= 0 ；否则，abs(x) 等于 -x 。

示例 1：

输入：n = 4, index = 2,  maxSum = 6
输出：2
解释：数组 [1,1,2,1] 和 [1,2,2,1] 满足所有条件。不存在其他在指定下标处具有更大值的有效数组。

示例 2：

输入：n = 6, index = 1,  maxSum = 10
输出：3

提示：

  • 1 <= n <= maxSum <= 109
  • 0 <= index < n

二分答案

二分搜索答案，答案的上界为maxSum，下界为1。若nums[index] = x能使数组的最小和小于等于maxSum，那么小于x的值也满足该条件，所以二分时X要上取整。而且整体应该是从大的值往小的逼近（high->low），即二分时high = mid-1,low = mid。

计算数组的最小和，已知数组长为n，令下标index处的值为x，那么应该把数组分为两个等差数列来计算（nums[0:index]和nums[index:end]）再减去x就是数组最小和。

又因为数组上每个值为正整数，所以最小值为1，需要判断两节数组的长度len和x的大小关系。

若len <= x，那么数组最小和为x-len+1到x的等差数列和；

否则存在多余的空位要填上1，数组和为1到x的等差数列和加上len-x。

算法时间复杂度为：\(O(\log(n))\)，n为数组maxSum的值。

class Solution {
    // 求一节数组的和
    public double compute(int n, long x) {
        double ans = 0;
        if (x > n) { 
            // len <= x 该节数组最小和为 x-len+1 到 x 的等差数列和
            ans = (x + x-n+1)*n/2;
        } else {
            // len > x 多余位置填1，数组 1111234..index
            ans = (1 + x)*x/2 + n-x;
        }
        return ans;
    }
    
    public int maxValue(int n, int index, int maxSum) {
        long low = 1;
        long high = maxSum;
        while (low < high) {
            // 计算mid往上取整
            long mid = (low + high + 1)>>1;
            // 整个数组的最小和 为两段数组的和 减去重复的x
        	double res = compute(index+1, mid)+compute(n-index, mid)-mid;
            if (res > maxSum) {
                high = mid-1;
            } else {
                low = mid;
            }
        }
        return (int)low;
    }
}

模拟

过程模拟，从 index 开始往两边扩充：维护一个 \([l,r]\)范围，每次往范围内每个位置 +1 ，通过这种方式维护一个向上生长的“三角形”。

  • 时间复杂度\(O(\sqrt{M})\)，其中\(M=maxSum-n\)。
  • 空间复杂度\(O(1)\)。
public int maxValue(int n, int index, int maxSum) {
    int l = index, r = index;
    int ans = 1;
    // 整个数组一开始全部填充为1，
    // rest记录先全部填充1后，剩下1的个数
    int rest = maxSum - n;
    while (l > 0 || r < n - 1) {
        int len = r - l + 1;
        if (rest >= len) {
            // 当前[l,r]范围全部+1
            rest -= len;
            ans++;
            // 往左右两边扩
            l = Math.max(0, l - 1);
            r = Math.min(n - 1, r + 1);
        } else {
            break;
        }
    }
    // 扩大到整个数组之后，剩余的值“雨露均沾”一下
    ans += rest / n;
    return ans;
}
