6238. 统计构造好字符串的方案数

难度中等

给你整数 zero ，one ，low 和 high ，我们从空字符串开始构造一个字符串，每一步执行下面操作中的一种：

  • 将 '0' 在字符串末尾添加 zero 次。
  • 将 '1' 在字符串末尾添加 one 次。

以上操作可以执行任意次。

如果通过以上过程得到一个 长度 在 low 和 high 之间（包含上下边界）的字符串，那么这个字符串我们称为 好 字符串。

请你返回满足以上要求的 不同 好字符串数目。由于答案可能很大，请将结果对 109 + 7 取余 后返回。

示例 1：

输入：low = 3, high = 3, zero = 1, one = 1
输出：8
解释：
一个可能的好字符串是 "011" 。
可以这样构造得到："" -> "0" -> "01" -> "011" 。
从 "000" 到 "111" 之间所有的二进制字符串都是好字符串。

示例 2：

输入：low = 2, high = 3, zero = 1, one = 2
输出：5
解释：好字符串为 "00" ，"11" ，"000" ，"110" 和 "011" 。

提示：

  • 1 <= low <= high <= 105
  • 1 <= zero, one <= low

爬楼梯换皮

class Solution:
    def countGoodStrings(self, low: int, high: int, zero: int, one: int) -> int:
        # 算每个长为i的字符串的个数
        # 定义f[i] 表示长为i的字符串的个数
        # f[i] = f[i-zero] + f[i-one]
        MOD = 10 ** 9 + 7
        f = [0] * (high+1)
        f[0] = 1 # 表示空串的方案数是1
        for i in range(1,high+1):
            if i >= one: f[i] = (f[i] + f[i-one]) % MOD
            if i >= zero: f[i] = (f[i] + f[i-zero]) % MOD
        return sum(f[low:]) % MOD  # 从low到末尾high

DFS（超时）

class Solution {
    public static final int MOD = (int)1e9+7;
    String zero0;
    String one1;
    int dp[];
    public int countGoodStrings(int low, int high, int zero, int one) {
        dp = new int[high+10];
        StringBuilder sb = new StringBuilder();
        for(int i = 0; i < zero; i++) sb.append("0");
        zero0 = sb.toString();
        sb = new StringBuilder();
        for(int i = 0; i < one; i++) sb.append("1");
        one1 = sb.toString();
        sb = new StringBuilder();
        return dfs(low,high,sb);
    }
    public int dfs(int low,int high,StringBuilder sb){
        int n = sb.toString().length();
        if(n > high) return 0;
        if(dp[n] != 0) return dp[n];
        int res = 0;
        if(n >= low && n <= high){
            res += 1;
        }
        sb.append(zero0);
        res += dfs(low,high,sb);
        res %= MOD;
        int tmp = sb.toString().length();
        sb.delete(tmp-zero0.length(),tmp);
        sb.append(one1);
        res += dfs(low,high,sb);
        res %= MOD;
        tmp = sb.toString().length();
        sb.delete(tmp-one1.length(),tmp);
        dp[n] = res;
        return res;
    }
}
