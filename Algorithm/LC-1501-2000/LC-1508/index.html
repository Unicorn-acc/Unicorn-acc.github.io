1508. 子数组和排序后的区间和

难度中等

给你一个数组 nums ，它包含 n 个正整数。你需要计算所有非空连续子数组的和，并将它们按升序排序，得到一个新的包含 n * (n + 1) / 2 个数字的数组。

请你返回在新数组中下标为 left 到 right （下标从 1 开始）的所有数字和（包括左右端点）。由于答案可能很大，请你将它对 10^9 + 7 取模后返回。

示例 1：

输入：nums = [1,2,3,4], n = 4, left = 1, right = 5
输出：13 
解释：所有的子数组和为 1, 3, 6, 10, 2, 5, 9, 3, 7, 4 。将它们升序排序后，我们得到新的数组 [1, 2, 3, 3, 4, 5, 6, 7, 9, 10] 。下标从 le = 1 到 ri = 5 的和为 1 + 2 + 3 + 3 + 4 = 13 。

示例 2：

输入：nums = [1,2,3,4], n = 4, left = 3, right = 4
输出：6
解释：给定数组与示例 1 一样，所以新数组为 [1, 2, 3, 3, 4, 5, 6, 7, 9, 10] 。下标从 le = 3 到 ri = 4 的和为 3 + 3 = 6 。

示例 3：

输入：nums = [1,2,3,4], n = 4, left = 1, right = 10
输出：50

提示：

  • 1 <= nums.length <= 10^3
  • nums.length == n
  • 1 <= nums[i] <= 100
  • 1 <= left <= right <= n * (n + 1) / 2

按照题意模拟

class Solution {
    private static final int MOD = (int)1e9+7;
    public int rangeSum(int[] nums, int n, int left, int right) {
        int[] sum = new int[n*(n+1)/2];
        for(int i = 0,k = 0; i < n; i++){
            int s = 0;
            for(int j = i; j < n; j++){
                s += nums[j];
                sum[k++] = s;
            }
        }
        Arrays.sort(sum);
        int ans = 0;
        for(int i = left-1; i < right; i++){
            ans = (ans + sum[i]) % MOD;
        }
        return ans;
    }
}

二分+前缀和

题解：https://leetcode.cn/problems/range-sum-of-sorted-subarray-sums/solution/onlogs-jie-fa-er-fen-shuang-zhi-zhen-by-newhar/

详细版：https://leetcode.cn/problems/range-sum-of-sorted-subarray-sums/solution/-by-mkdir700-l90o/

思路

1. 求出新数组的第 k 项，也就是找出原数组中第 k 小的区间和 x。

2. 求出新数组的前 k 项和 f(k)。

3. 算出答案。

根据 \(f(k)\)的定义，题目的答案就是 \(f(right) - f(left-1)\)（不要忘了取模）。

class Solution {
    int n;
    long[] sum;
    long[] ss;

    public int rangeSum(int[] nums, int _n, int left, int right) {
        // 2.二分法+二阶前缀和
        n = _n;
        // 前缀和
        sum = new long[n + 1];
        for (int i = 0; i < n; i++) {
            sum[i + 1] = sum[i] + nums[i];
        }
        // 二阶前缀和
        ss = new long[n + 1];
        for (int i = 0; i < n; i++) {
            ss[i + 1] = ss[i] + sum[i + 1];
        }
        // sum子数组和数组排序后的两个前缀和相减就是答案
        return (int) ((F(right) - F(left - 1)) % (long) 1000000007);
    }

    // 求sum子数组和数组对应的index项的前缀和
    private long F(int index) {
        // 获取sum子数组和排序后第index位置的数字num
        // res为返回的前缀和结果,cnt为前面与index项不同的数目
        int num = getKth(index), cnt = 0;
        long res = 0;
        for (int i = 0, p = 1; i < n; i++) {
            // 这里搜索到>=num就截止,表明已经统计了所有严格<num的数字
            // 这里sum[p]-sum[i]实质上是求该行的上界,然后前面的全部加入
            while (p <= n && sum[p] - sum[i] < num) p++;
            // 其和加入结果
            res += ss[p - 1] - ss[i] - sum[i] * (long) (p - i - 1);
            // 统计数目
            cnt += p - i - 1;
        }
        // 与num相同的与不相同的相加就是结果
        return res + (long) (index - cnt) * num;
    }

    // 获取sum子数组和排序后第index位置的数字(这个类似于Q378 有序矩阵中第 K 小的元素)
    private int getKth(int index) {
        int l = 0, r = (int) 1e5;
        while (l < r) {
            int m = l + (r - l) / 2;
            int cnt = getCnt(m);
            // 若sum子数组和数组中<=m元素个数的大于等于index,向左查找(含)
            if (cnt >= index) r = m;
                // 若sum子数组和数组中<=m元素个数的小于index,向右查找(不含)
            else l = m + 1;
        }
        return l;
    }

    // 求若sum子数组和数组中<=m元素个数(倒立阶梯每一行求和)
    private int getCnt(int m) {
        int res = 0;
        for (int i = 0, p = 1; i < n; i++) {
            while (p <= n && sum[p] - sum[i] <= m) p++;
            res += p - i - 1;
        }
        return res;
    }
}
