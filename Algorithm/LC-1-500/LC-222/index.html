222. 完全二叉树的节点个数

难度中等

给你一棵 完全二叉树 的根节点 root ，求出该树的节点个数。

完全二叉树 的定义如下：在完全二叉树中，除了最底层节点可能没填满外，其余每层节点数都达到最大值，并且最下面一层的节点都集中在该层最左边的若干位置。若最底层为第 h 层，则该层包含 1~ 2h 个节点。

示例 1：

输入：root = [1,2,3,4,5,6]
输出：6

示例 2：

输入：root = []
输出：0

示例 3：

输入：root = [1]
输出：1

提示：

  • 树中节点的数目范围是[0, 5 * 104]
  • 0 <= Node.val <= 5 * 104
  • 题目数据保证输入的树是 完全二叉树

进阶：遍历树来统计节点是一种时间复杂度为 O(n) 的简单解决方案。你可以设计一个更快的算法吗？

DFS

class Solution {
    public int countNodes(TreeNode root) {
        if(root == null) return 0;
        int left = countNodes(root.left);
        int right = countNodes(root.right);
        return left + right + 1;
    }
}

根据二叉树性质简化遍历次数

一棵完全二叉树：除最后一层外，其余层全部铺满；且最后一层向左停靠

  • 如果根节点的左子树深度等于右子树深度，则说明左子树为满二叉树

  • 如果根节点的左子树深度大于右子树深度，则说明右子树为满二叉树

  • 如果知道子树是满二叉树，那么就可以轻松得到该子树的节点数目：(1<<depth) - 1; // depth为子树的深度；为了加快幂的运算速度，可以使用移位操作符
  • 接着我们只需要接着对另一子树递归即可

  • 时间复杂度为O(logn)，空间复杂度为O(1)【不考虑递归调用栈】

class Solution {
    public int countNodes(TreeNode root) {
        if(root == null)    return 0;
        int left = height(root.left);
        int right = height(root.right);

        //(1 << left) : 2^left次方
        return left == right ? 
                (1 << left) + countNodes(root.right) : // 左子树 是 perfect 二叉树
                (1 << right) + countNodes(root.left); // 右子树 是 perfect 二叉树

    }

    // 计算 complete 二叉树的 高度
    private int height(TreeNode root){
        int res = 0;
        while(root != null){
            root = root.left;
            res++;
        }
        return res;
    }
}

二分查找

根据完全二叉树的性质，我们可以清楚的知道：总节点数 = 倒数第二层以上的节点数 + 最后一层的节点数

  • 除最后一层外，这棵树为满二叉树，节点数为：2^depth_prev - 1，depth_prev为倒数第二层树的深度
  • 最后一层的节点数的范围是 [1, 2^depth_prev]；并且依次靠左排列
  • 所以现在的问题就转换为判断最后一层节点数

先不说怎么判断一个节点是否存在，假设这个方法为is_exist()，我们先了解一下二分法如何使用：

最后一层节点情况： 1代表非空节点，0代表空节点

所以现在的问题就可以简单的看成最后一个1的索引位置。

接下来是如何判断最后一层某个节点是否存在，也就是is_exist()函数：

给定最后一层某节点的位置索引 index，将他和分界线比大小，就可以判断该节点在左子树还是右子树，例如：现在查找6这个节点，索引为3，大于分界线2，所以6在右子树；对右子树重复操作即可，剩下的工作就交给迭代了~~~

class Solution {
    public int countNodes(TreeNode root) {
        if(root == null)    return 0;
        int d = depth(root);
        /*
            对于 perfect binary tree, 如果 深度为 d （定义：根节点的深度为1）
            节点总数 2^d - 1
            当前层 d 的节点总数 2^(d-1)
        */

        // 二分查找 [start, end] 中 最后一个 exist == true 的 位置
        int start = 1, end = (1 << (d-1)); 
        while(start <= end){
            int mid = (start + end) >>> 1;
            if(exist(root, mid, d)){
                start = mid + 1; // 再向右 尝试一下
            }else{
                end = mid - 1;
            }
        }
        // 跳出循环后 [1...end] 都是 非空节点
        // 前 d-1 层的 节点总数 (1 << (d-1)) - 1
        return (1 << (d-1)) - 1 + end;
    }

    private int depth(TreeNode root){
        int res = 0;
        while(root != null){
            res++;
            root = root.left;
        }
        return res;
    }

    // 判断 以 root 为根的 complete 二叉树的最后一层 index 位置是否有节点
    // d 表示 以 root 为根的 complete 二叉树的 层数
    // 则 1 <= index <= 2^(d-1) 从1开始计数
    // root != null
    private boolean exist(TreeNode root, int index, int d){
        while(--d != 0){ // 上来先-1，方便计算某一层节点数，之后是循环变量-1
            int mid = (1 << (d)) >>> 1;
            if(index > mid){
                root = root.right;
                index -= mid; // index 从1计数
            }else{
                root = root.left;
            }
        }
        return root != null;
    }
}
